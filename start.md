
## compile
```
sudo apt-get install ros-noetic-jsk-recognition
sudo apt-get install ros-noetic-jsk-common-msgs
sudo apt-get install ros-noetic-jsk-rviz-plugins

export CC=gcc-11
export CXX=g++-11

```


## run

```
# demo
roslaunch erasor run_erasor_in_your_env_vel16.launch

# biaozhi
roslaunch erasor run_erasor_biaozhi.launch
```
